Let's Go

Code Porting

Team or Solo

Date : March 11

Event Description
Hash 2018 presents you with an all new opportunity to showcase your cross-language coding prowess. The challenge is to port a given snippet of code/program from language X to language Y. Your code should perform the same basic function as the given program and you're all set. Expect challenges in C, C++, Java and Python.

Rules and regulations
The maximum team size allowed is two.
Each team will be given one system to code.
The participants are not allowed to use the internet or any other external sources during the event.
College IDs are compulsory.
Members of a team needn’t be from the same institution.
A participation certificate will be issued for all the participants of the event.
The decisions of judges will be final.
Violation of any rules will lead to immediate disqualification of the team.

Registration fee: ₹ 50 per head.
